Como comenzar

Para comenzar a utilizar este pequeño framework deberás clonar o descargar el repositorio y posteriormente dirigir tu servidor web a la carpeta public, en la cual se encuentra el punto de entrada a la aplicación. Deberas configurar tu archivo .htacces como el ejemplo contenido en el repositorio o de lo contrario configurar el servidor que estes utilizando para poder trabajar con las url friendly que maneja el framework.

Para comprobar que todo esta funcionando correctamente simplemente deberás acceder a la dirección http://tudireccion/test/getVar/1 y si en tu navegador observas simplemente "1" has concluido la configuración básica.

El framework trabaja bajo la convención "dominio.com/controlador/accion" por lo cual para añadir nuevas rutas a tu aplicación bastara con añadir nuevos controladores y nuevas acciones en estos, siguiendo el ejemplo existente en la carpeta controllers.
